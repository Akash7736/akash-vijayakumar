Manipulator Project – Robot Mechanical Systems

Overview

This course project, completed as part of the Robot Mechanical Systems course, focuses on the modeling and control of a planar robotic manipulator. The project explores the full pipeline from defining the robot in MuJoCo XML to implementing forward and inverse kinematics in Python and running interactive simulations.

The system is built around a 3-DOF manipulator model, where the goal is to accurately position the end-effector while respecting joint limits and dynamic constraints. The codebase combines analytical kinematics with physics-based simulation to visualize the manipulator motion and validate the algorithms.

Technical Details

Domain: Robotics & Manipulators Tools: Python, MuJoCo Focus: Kinematics & Simulation
  • Manipulator modeled in MuJoCo using XML scene descriptions (main.xml, main_deb.xml).
  • Forward and inverse kinematics implemented in Python (dim.py, ik.py).
  • Interactive simulation and visualization driven by run.py.
  • Mesh assets used to render the manipulator links and environment.

What I Learned

  • Formulating forward and inverse kinematics for a multi-DOF manipulator.
  • Integrating analytical robotics models with a physics-based simulator (MuJoCo).
  • Designing clean Python modules for robotics experimentation.
