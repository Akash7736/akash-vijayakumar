Virtual RobotX Challenge 2023

Virtual RobotX Challenge 2023 is a competition organized by Roborx, IIT Madras. The challenge involves tasks such as maneuvering, docking, and collision avoidance to be performed by the autonomous surface vehicle.

Project Overview

This project focuses on developing an autonomous surface vehicle capable of performing complex tasks in a virtual environment. The vehicle is equipped with advanced sensors and control systems to navigate and complete tasks efficiently.

Key Features

  • Advanced maneuvering and docking capabilities
  • Collision avoidance using Lidar and Stereo Vision
  • Integration with ROS1 and rosserial for control
  • Utilization of yolov8 for object detection

Technical Specifications

  • Platform: Catamaran
  • Sensors: Lidar, Stereo Vision
  • Software: ROS1, rosserial, yolov8
